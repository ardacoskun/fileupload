### \***\*📲 About\*\***

This is a MERN stack file upload and download project.Users can upload their files if they have permitted types. Users can display,delete and download their files.

### 🔎 Technologies

- Node.js
- Express
- React
- Multer
- Jest
- Axios
- MongoDB

---

## ⚙️\***\*Installation\*\***

1. Clone this repo, install dependencies.

git clone https://github.com/ardacoskun/fileupload.git

```makefile
cd client
npm install

cd server
npm install
```
---

2- Create .env file like;

```makefile
PORT = <port number>
MONGO_URI = <MongoDB Connection Uri>
```

3- Build and run project;

```makefile
cd client
npm start

cd server
npm start
```

4- Run Tests;

```makefile
cd client
npm test

cd server
npm test
```
